# QA Login Tests - SwagLabs (SauceDemo)

### 📌 Sobre o projeto

Este projeto tem como objetivo realizar testes manuais e automatizados na funcionalidade de login do sistema **SwagLabs (SauceDemo)**, uma aplicação web de e-commerce utilizada para prática de testes de software.

🔗 Sistema testado: https://www.saucedemo.com/

---

### 🎯 Objetivo

Validar o funcionamento da tela de login, garantindo:

- Funcionamento correto do fluxo principal
- Tratamento de entradas inválidas
- Verificação de cenários de borda
- Validação básica de segurança
- Acessibilidade no fluxo

---

### 🔬 Tipos de testes realizados

- ✅ Testes funcionais
- ❌ Testes negativos
- ⚠️ Testes de borda
- 🔐 Testes de segurança
- 👁️‍🗨️ Testes de acessibilidade
- 📱 Testes de responsividade

### 🧠 Análise e Conclusão de QA

Até o momento foram realizados testes manuais e o sistema apresentou funcionamento adequado no fluxo principal de login, todavia foram identificadas melhorias de severidade baixa e média, como validação de entrada envolvendo espaços em branco e na legibilidade da mensagem de erro.
